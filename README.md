# 📌 Requirement

```bash
$ npm install -g expo-cli
$ npm install -g typescript
```

<br/>

# 📌 Start

```bash
$ expo start
```

```plaintext
1. 앱스토어에서 'Expo Go' 앱 설치
2. 위 명령어 실행 후, 카메라로 QR코드 인식 OR a 또는 i 키 눌러서 시뮬레이터로 확인
3. Expo Go에서 앱 실행 확인
(코드 실행중인 노트북과 Expo Go를 실행중인 모바일의 네트워크가 동일해야함)
(ex. 동일한 와이파이 쓰던가 핫스팟으로 연결하던가)
```
